# 权限管理功能 - 验证清单

- [ ] 普通用户（USER）登录后，菜单中不显示用户管理和图书管理选项
- [ ] 普通用户（USER）尝试通过URL直接访问用户管理页面时，应被阻止并跳转到合适页面
- [ ] 普通用户（USER）尝试通过URL直接访问图书管理页面时，应被阻止并跳转到合适页面
- [ ] 管理员（ADMIN）登录后，菜单中显示用户管理和图书管理选项
- [ ] 管理员（ADMIN）可以正常访问用户管理页面
- [ ] 管理员（ADMIN）可以正常访问图书管理页面
- [ ] 超级管理员（SUPER_ADMIN）登录后，菜单中显示用户管理和图书管理选项
- [ ] 超级管理员（SUPER_ADMIN）可以正常访问用户管理页面
- [ ] 超级管理员（SUPER_ADMIN）可以正常访问图书管理页面
- [ ] 权限控制逻辑在前端路由层面实现，确保安全性
- [ ] 权限控制逻辑集中管理，便于后续扩展和修改
- [ ] 权限判断逻辑在用户登录后立即执行，不影响页面加载速度
